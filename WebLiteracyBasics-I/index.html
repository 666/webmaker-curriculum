Web Literacy Basics I

Made by the #TeachTheWeb team with special thanks to Mouse.

In this module, learners begin to find out about the foundations of the web. Discovering what makes the web a unique place to find and gather information, they'll learn about how the web is constructed.

  • Learning Objectives

    • Locate relevant information by using search engines
    • Compare and contrast information found using multiple searches
    • Decide on the trustworthiness of information by identifying key indicators

  • Tools

    • the Browser
    • X-Ray Goggles
